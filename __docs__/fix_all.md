# Fix All - Linting y Type Checking

Ejecuta los siguientes comandos y resuelve todos los problemas encontrados:

```bash
uvx ruff format
uvx ruff check
uvx ty check
```

## Instrucciones

1. **ruff format**: Formatea automáticamente el código. Si hay archivos modificados, revisa los cambios.
2. **ruff check**: Ejecuta el linter. Si hay errores, corrígelos uno por uno.
3. **ty check**: Verifica tipos. Corrige cualquier error de tipado.

Si algún comando falla, arregla los problemas y vuelve a ejecutar hasta que todos pasen.
